# Recruiter Summary - RACF / zOS Security Evidence

## What this lab proves

This lab proves practical exposure to RACF in a real z/OS-style 3270 environment.

The candidate can:

- Use TSO/RACF inquiry commands.
- Review RACF user attributes.
- Identify privileged accounts.
- Review RACF group membership.
- Explain security risk in an audit/SOC style.
- Document technical evidence clearly.

## Key RACF evidence

- `LISTUSER IBMUSER` showed `SPECIAL` and `OPERATIONS`.
- `LISTGRP SYS1` showed group ownership, subgroups, and connected users.
- Troubleshooting evidence was kept for a `LISTDSD` command syntax issue.
