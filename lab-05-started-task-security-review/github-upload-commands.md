# GitHub Upload Commands

If using Git Bash on Windows, first unzip the generated lab folder into your repository.

Example:

```bash
cd /c/Carrera_Ciberseguridad/06_Portfolio_GitHub/mainframe-racf-security-evidence

git status
git add lab-05-started-task-security-review
git commit -m "Add started task security review lab"
git push
```

If the repository has a different local path, change the `cd` command accordingly.
