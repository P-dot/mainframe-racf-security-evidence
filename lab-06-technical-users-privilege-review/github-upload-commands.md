# GitHub Upload Commands

Download and unzip this lab into your repository, then run:

```bash
cd /c/Carrera_Ciberseguridad/06_Portfolio_GitHub/mainframe-racf-security-evidence

git status
git add lab-06-technical-users-privilege-review
git commit -m "Add RACF technical users privilege review lab"
git push
```

If the lab folder is not yet inside the repository, copy or unzip it into the repository root first.
