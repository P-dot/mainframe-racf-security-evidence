# GitHub Upload Commands

From Git Bash, after extracting the ZIP into your repository:

```bash
cd /c/Carrera_Ciberseguridad/06_Portfolio_GitHub/mainframe-racf-security-evidence

git status
git add lab-07-zos-unix-omvs-security-baseline
git commit -m "Add zOS UNIX OMVS security baseline lab"
git push
```

If the lab folder already exists and you are updating it:

```bash
git status
git add lab-07-zos-unix-omvs-security-baseline
git commit -m "Update zOS UNIX OMVS security baseline evidence"

```
