# GitHub Upload Commands

Download and unzip this lab into your repository, then run:

```bash
cd /c/Carrera_Ciberseguridad/06_Portfolio_GitHub/mainframe-racf-security-evidence

git status
git add lab-08-racf-dataset-protection-baseline
git commit -m "Add RACF dataset protection baseline lab"
git push
```

If the lab was downloaded as a ZIP, unzip it first into the repository root so that the folder is:

```text
mainframe-racf-security-evidence/lab-08-racf-dataset-protection-baseline
```
